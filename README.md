# HR-Analytic-Excel-
HR Analytics Dashboard Excel Project

📌 Project Overview : 
This project is an HR Analytics Dashboard created in Microsoft Excel.

The dashboard helps analyze employee data and gives useful insights about:
Employee count
Department-wise distribution
Employee performance
Hiring trends
Gender distribution
Employee engagement and satisfaction
Training cost analysis

The dashboard is interactive and easy to understand using slicers and charts.

🎯 Objective : 

The main goal of this project is to:
Analyze HR data visually
Track employee performance
Understand hiring patterns
Compare department statistics
Improve decision-making using data

🛠️ Tools Used : 
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Timeline Filter
Conditional Formatting
Dashboard Design

📊 Dashboard Features : 
1. KPI Cards

The dashboard contains important HR metrics:
Total Employees: 3148
Average Engagement Score: 2.94
Average Satisfaction Score: 3.02
Training Cost: 17.65M

2. Department-wise Employee Analysis : 

Shows the total number of employees in each department such as:
Production
IT/IS
Sales
Software Engineering
Admin Offices
Executive Office

This helps identify departments with higher workforce distribution.

3. Employee Performance Analysis : 
Employees are categorized into:
Fully Meets
Exceeds
Needs Improvement
PIP

This helps HR understand employee performance levels.

4. Hiring Trend Analysis : 

The line chart shows yearly hiring trends from 2018 to 2023.

It helps identify:
Hiring growth
Recruitment decline
Workforce planning trends

5. Gender Distribution : 
The pie chart shows gender diversity in the organization.

Female: 56%
Male: 44%

6. Interactive Filters :
The dashboard includes slicers for:
Gender
Department Type
Performance Score
Start Date Timeline

Users can filter data dynamically for better analysis.

📈 Key Insights : 
Production department has the highest number of employees.
Most employees fall under the “Fully Meets” performance category.
Female employees are slightly higher than male employees.
Hiring increased significantly after 2018.
Training cost is an important organizational investment.
